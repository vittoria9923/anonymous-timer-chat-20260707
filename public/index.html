초대 없이 누구나 들어올 수 있는 기본 공개방입니다.

01

Random Identity

이름과 아바타는 자동 생성됩니다. 가입, 이메일, 비밀번호가 필요 없습니다.

02

Image Message

작은 사진을 바로 공유하고, 텍스트와 동일한 만료 규칙으로 사라지게 합니다.

03

Expiry Timer

각 메시지는 서버 타이머에 의해 삭제되고 연결된 모든 클라이언트에 반영됩니다.

Render + Netlify

서버는 Render, 프론트는 Netlify.

Socket.IO 서버는 Render의 Node 웹 서비스에서 실행하고, Netlify는 정적 프론트엔드를 제공합니다. `public/config.js`의 서버 주소만 Render URL로 맞추면 분리 배포가 가능합니다.
